## html 요소

```html
html의 6가지 버전 (1,2,3,4,xhtml,5)중 어떤 버전으로 보여주어여 하는지 명시하는것
<!DOCTYPE html>
/* 문서의 기본 언어를 설정 */
<html lang="ko">
  /* 문서의 정보 영역 */
  <head>
    /* 문자의 인코딩 방식을 설정 */
    <meta charset="UTF-8" />
    <meta name="author" content="제작자의 이름" />
    <meta name="description" content="사이트 설명" />
    <title>페이지의 제목</title>
  </head>
  /* 문서의 구조 영역 */
  <body></body>
</html>
```
