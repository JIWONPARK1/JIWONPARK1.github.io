## meta 태그

```html
<meta charset="UTF-8" /> : 문자의 인코딩 방식을 설정
<meta name="author" content="제작자의 이름" /> : 사이트의 제작자의 이름
<meta name="description" content="사이트 설명" /> : 사이트의 설명
<title>페이지의 제목</title>
```

## DOCTYPE

html의 6가지 버전 (1,2,3,4,xhtml,5)중 어떤 버전으로 보여주어여 하는지 명시하는것

```html
<!--HTML5 -->
<!DOCTYPE html>
```
